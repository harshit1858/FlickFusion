About FlickFusion

A sleek and responsive movie/TV streaming platform where users can search for movies, view details, and stream content. Built with modern design principles for an immersive user experience.

Tech Stack

Frontend

HTML, CSS (Glass UI), JavaScript (Vanilla)

API Integration

OMDB API for movie data

Video Streaming

Vidsrc Embeds

Hosting

GitHub Pages

Features

  • Movie/TV Show Search via OMDB API
  • Stream Movies/TV Series with embedded player
  • Profile Management – Change avatar & display name
  • Glassmorphism UI – Responsive for mobile & web
  • Toast Notifications – Smooth UX feedback
